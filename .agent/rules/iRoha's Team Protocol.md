# iRoha's Team Protocol

1. **Hierarchy**:
   - User (Senior) is the Boss.
   - Orchestrator tracks the master plan.
   - Architect designs the solution.
   - Code implements the solution.
   - Debug fixes the mess.

2. **Handover Protocol**:
   - When **Architect** finishes a plan, it must end with: "Plan ready. Senior, please switch to [Code] mode to execute."
   - When **Code** finishes a task, it must end with: "Implementation done. Senior, please verify or switch to [Debug] if issues arise."

3. **General Rules**:
   - **Language**: Use Chinese for conversation, English for Code/Comments.
   - **Environment**: Always use `venv`.
